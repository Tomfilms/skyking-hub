📝 RULE

Color = LENGTH only. Owner and destination go on printed text labels. Colored tape goes on both ends of every cable.

🏷️ SKYKING ID FORMAT: SK-[CAT]-[NUM]

Examples:

SK-CAM-001 SK-CBL-001 SK-PWR-001 SK-MON-001 SK-AUD-001 SK-NET-001 SK-CASE-001 SK-MISC-001

🐧 LINUX TERMINAL BASICS (GreenDuck)

Open terminal: Ctrl+Alt+T on GreenDuck

Copy/paste IN terminal: Ctrl+Shift+C and Ctrl+Shift+V (not regular Ctrl+C)

Go to your home folder
cd ~
List files in current folder
ls
List files with details
ls -la
See what USB devices are plugged in
lsusb
Run the label maker app
python3 ~/scripts/labelmaker.py
Run the gear database app
python3 ~/scripts/geardatabase.py

🖨️ PT-2100 LABEL COMMANDS

Test print — 2 lines
ptouch-print --font "DejaVu Sans" --fontsize 14 --print "LINE ONE LINE TWO"
Preview label first (creates label.png)
ptouch-print --font "DejaVu Sans" --fontsize 14 --writepng label.png "LINE ONE LINE TWO"

⚠️ Preview and print are separate commands — don't combine with &&
